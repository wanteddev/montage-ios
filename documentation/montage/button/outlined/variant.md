# Button.Outlined.Variant 

Outlined 스타일 버튼의 변형을 정의합니다.

```swift
enum Variant
```

## Overview 

- .primary: 기본 강조 스타일 - 브랜드 컬러의 테두리를 사용
- .secondary: 보조 강조 스타일 - 브랜드 컬러보다 덜 강조된 스타일
- .assistive: 보조 스타일 - 중요도가 낮은 보조 액션에 사용
