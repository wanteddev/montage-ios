# Menu.Variant 

메뉴의 표시 형태를 정의하는 열거형입니다.

```swift
enum Variant
```

## Overview 

- normal: 기본 메뉴 형태로, 선택 표시 없이 항목만 표시
- radio: 라디오 버튼 형태로, 단일 항목만 선택 가능
- checkbox: 체크박스 형태로, 다중 항목 선택 가능
