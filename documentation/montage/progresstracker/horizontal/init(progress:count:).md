# init(progress:count:) 

라벨 없는 수평 진행 추적기를 초기화합니다.

```swift
@MainActor
init(
    progress: Binding<Int>,
    count: Int
)
```

## Parameters 

progress

현재 진행 중인 단계 (1부터 시작)

count

전체 단계 수
