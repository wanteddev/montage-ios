# init(title:onTap:) 

셀 컴포넌트를 초기화합니다.

```swift
@MainActor
init(
    title: String,
    onTap: (() -> Void)? = nil
)
```

## Parameters 

title

셀에 표시할 제목 텍스트

onTap

셀을 탭했을 때 실행할 클로저
