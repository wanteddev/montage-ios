# Chip 

다양한 형태의 칩(Chip) 컴포넌트를 위한 네임스페이스입니다.

```swift
enum Chip
```

## Overview 

Chip은 Montage 디자인 시스템에서 필터링, 선택, 작은 액션 등을 위한 콤팩트한 UI 요소를 제공하는 열거형 네임스페이스입니다.

주요 기능:

- 필터링 기능을 위한 필터 칩 (Filter)
- 액션 실행을 위한 액션 칩 (Action)

> **See Also**
>
> Chip.Filter, Chip.Action
