Initializer

# init(spacing:lineSpacing:) 

FlowLayout을 초기화합니다.

```swift
init(
    spacing: CGFloat? = nil,
    lineSpacing: CGFloat = 10.0
)
```

## Parameters 

spacing

항목 간 수평 간격. nil인 경우 시스템 기본 간격을 사용합니다. (기본값: nil)

lineSpacing

줄 간 수직 간격 (기본값: 10.0)
