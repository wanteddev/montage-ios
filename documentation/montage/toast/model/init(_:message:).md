# init(_:message:) 

Toast 모델을 초기화합니다.

```swift
init(
    _ variant: Toast.Variant = .normal(),
    message: String
)
```

## Parameters 

variant

토스트 메시지의 스타일 (기본값: .normal())

message

토스트에 표시할 메시지 텍스트
