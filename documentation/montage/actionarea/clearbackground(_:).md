# clearBackground(_:) 

배경을 투명하게 설정합니다.

```swift
@MainActor
func clearBackground(_ clearBackground: Bool = true) -> ActionArea
```

## Parameters 

clearBackground

배경 투명 여부, 기본값은 true

## Return Value 

수정된 ActionArea 인스턴스

## Discussion 

이 수정자를 사용하면 그라데이션 배경이 숨겨지고 투명한 배경이 표시됩니다.
