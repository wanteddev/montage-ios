# TextInput 

텍스트 입력 관련 컴포넌트를 위한 네임스페이스입니다.

```swift
enum TextInput
```

## Overview 

TextInput은 Montage 디자인 시스템에서 텍스트 입력 관련 컴포넌트를 제공하는 열거형 네임스페이스입니다. 단일 라인 입력을 위한 TextField와 여러 줄 입력을 위한 TextArea 등을 포함합니다.

> **See Also**
>
> TextInput.TextField, TextInput.TextArea
