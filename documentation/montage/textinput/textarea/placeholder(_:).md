# placeholder(_:) 

텍스트 영역에 입력된 텍스트가 없을 때 표시할 플레이스홀더를 설정합니다.

```swift
@MainActor
func placeholder(_ placeholder: String?) -> TextInput.TextArea
```

## Parameters 

placeholder

표시할 플레이스홀더 텍스트

## Return Value 

수정된 텍스트 영역 인스턴스
