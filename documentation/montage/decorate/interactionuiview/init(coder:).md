Initializer

# init(coder:) `Deprecated`

인터페이스 빌더에서 사용할 때 호출되는 초기화 메서드입니다.

```swift
@MainActor
required init?(coder: NSCoder)
```

> **Deprecated**
>
> 이 클래스는 더 이상 사용되지 않으며 향후 버전에서 제거될 예정입니다. 대신 SwiftUI 기반의 InteractionView를 사용하세요.

## Parameters 

coder

디코더 객체

## Return Value 

디코딩된 상호작용 장식 뷰 인스턴스, 실패 시 nil
