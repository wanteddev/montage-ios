# init(_:variant:) 

구분선 컴포넌트를 초기화합니다.

```swift
@MainActor
init(
    _ axis: Axis,
    variant: Variant = .normal
)
```

## Parameters 

axis

구분선의 방향 (.horizontal 또는 .vertical)

variant

구분선의 두께 변형, 기본값은 .normal

## Return Value 

구성된 구분선 인스턴스
