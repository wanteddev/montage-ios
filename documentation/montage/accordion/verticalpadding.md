# Accordion.VerticalPadding 

아코디언의 상하 여백을 나타내는 열거형입니다.

```swift
enum VerticalPadding
```

## Overview 

아코디언 헤더의 상하 패딩 값을 설정합니다.

- small: 좁은 여백 (8pt)
- medium: 중간 여백 (12pt)
- large: 넓은 여백 (16pt)
