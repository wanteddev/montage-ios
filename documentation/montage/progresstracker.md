# ProgressTracker 

단계별 진행 상태를 추적하고 표시하는 컴포넌트 타입을 그룹화하는 네임스페이스입니다.

```swift
enum ProgressTracker
```

## Overview 

ProgressTracker는 다단계 프로세스의 진행 상태를 시각적으로 표시하는 컴포넌트를 제공합니다. 수평 및 수직 레이아웃을 지원하며, 각 단계의 완료/활성/비활성 상태를 구분하여 표시합니다.

> **Note**
>
> 각 ProgressTracker컴포넌트는 개별 구현체로 제공되며, 이 열거형은 네임스페이스 역할만 합니다.

> **See Also**
>
> ProgressTracker.Horizontal, ProgressTracker.Vertical
