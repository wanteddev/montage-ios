# Decorate 

UI 요소의 장식적 특성을 관리하는 네임스페이스입니다.

```swift
enum Decorate
```

## Overview 

Decorate는 Montage 디자인 시스템에서 UI 요소의 시각적 장식 요소를 정의하고 관리하는 열거형 네임스페이스입니다.

주요 기능:

- 사용자 상호작용에 따른 시각적 피드백 (Interaction)
