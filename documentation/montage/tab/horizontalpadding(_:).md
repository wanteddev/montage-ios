# horizontalPadding(_:) 

탭 컴포넌트의 좌우 여백 사용 여부를 설정합니다.

```swift
@MainActor
func horizontalPadding(_ horizontalPadding: Bool = true) -> Tab
```

## Parameters 

horizontalPadding

좌우 여백 사용 여부 (기본값: true)

## Return Value 

수정된 Tab 인스턴스

## Discussion 

> **Note**
>
> 기본값은 false입니다. true로 설정 시 좌우에 20pt 여백이 적용됩니다.
