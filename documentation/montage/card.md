# Card 

카드 형태의 UI 컴포넌트를 제공하는 네임스페이스입니다.

```swift
enum Card
```

## Overview 

다양한 형태의 카드 UI를 제공하여 정보를 시각적으로 구조화된 방식으로 표시합니다. 썸네일, 제목, 설명 등을 포함한 컨텐츠를 카드 형태로 보여줍니다.
