# Skeleton.Length 

스켈레톤 요소의 길이 비율을 지정하는 열거형입니다.

```swift
enum Length
```

## Overview 

텍스트 스켈레톤에서 각 라인의 길이를 상대적으로 지정하는 데 사용됩니다.

**사용 예시**:

```swift
.text(lengths: [._100, ._75, ._50], lineNumber: 3)
```
