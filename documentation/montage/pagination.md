# Pagination 

페이지네이션 컴포넌트 타입을 그룹화하는 네임스페이스입니다.

```swift
enum Pagination
```

## Overview 

Pagination은 다양한 페이지네이션 컴포넌트를 포함하는 상위 열거형으로, 점 형태의 페이지 표시기(Dot)와 카운터 형태의 페이지 표시기(Counter)를 제공합니다.

> **Note**
>
> 각 페이지네이션 컴포넌트는 개별 구현체로 제공되며, 이 열거형은 네임스페이스 역할만 합니다.

> **See Also**
>
> Pagination.Dot, Pagination.Counter
