# init(text:icon:isNegative:isSelected:) 

아이템 초기화

```swift
init(
    text: String,
    icon: Icon? = nil,
    isNegative: Bool = false,
    isSelected: Bool = false
)
```

## Parameters 

text

icon

아이템 아이콘 (기본값: nil)

isNegative

부정적 상태 여부 (기본값: false)

isSelected

선택 여부 (기본값: false)
