# Select.Variant.multiple(render:overflow:menuPrimaryButtonTitle:) 

다중 선택 모드

```swift
case multiple(
    render: Render = .text,
    overflow: Bool = false,
    menuPrimaryButtonTitle: String
)
```

## Parameters 

render

선택된 항목 표시 방식 (텍스트 또는 칩), 기본값은 .text

overflow

선택된 항목이 여러 줄로 표시되는지 여부, 기본값은 false

menuPrimaryButtonTitle

확인 버튼 제목
