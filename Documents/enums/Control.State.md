**ENUM**

# `Control.State`

```swift
public enum State
```

컨트롤의 상태를 정의하는 열거형입니다.

- unchecked: 선택되지 않은 상태
- checked: 선택된 상태
- indeterminate: 중간 상태 (checkbox에서만 지원)

## Cases
### `unchecked`

```swift
case unchecked
```

### `checked`

```swift
case checked
```

### `indeterminate`

```swift
case indeterminate
```
